# (SQL | Java | Python)

## Overview
This repository contains three mini projects using SQL, Core Java, and Python.

## 1. Retail Insights (SQL)
- Designed a database for inventory management  
- Identified **expiring products** and **dead stock**  
- Analyzed **revenue by category**  

---

## 2. FinSafe Transaction Validator (Java)
- Built a console-based wallet system  
- Prevented overdraft using **custom exception**  
- Maintained last 5 transactions  

---
## 3. Social Media Content Sanitizer (Python)
- Filtered posts with banned words  
- Extracted links and saved to file  
- Generated moderation report
