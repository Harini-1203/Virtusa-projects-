# Social Media Content Sanitizer

## Description
This project is a Python script that checks social media posts for inappropriate words and links.

## problem statement
Build a Content Moderator Script that scans a list of posts, replaces "Banned Words" with
asterisks (***), and extracts all web links for security clicking.


## Output
- Cleaned posts
- Total posts screened
- Number of cleaned and blocked posts
- User flag count
- Extracted links saved in links.txt
